# 概述
UI-PROTOTYPE 提供了基本的编译功能，开箱即用。

# 使用方法
## 编译全部文件
```sh
npm run compile
```

## 编译 TS
```sh
npm run compile-ts
```

## 编译样式文件
```sh
npm run compile-style
```

## 编译本地化文件
```sh
npm run compile-locale
```

## 编译资源文件
```sh
npm run compile-assets
```

## 编译 HTML 模板
```sh
npm run compile-pug
```

参考[使用 VSCode 开发](../vscode)章节。
