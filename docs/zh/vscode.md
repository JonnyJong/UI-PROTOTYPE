# 使用 VSCode 开发
推荐使用 VSCode 开发 UI-PROTOTYPE。

下面是推荐安装的插件。

| 名称        | 扩展 ID                  | 说明               |
| ----------- | ------------------------ | ------------------ |
| Run on Save | `emeraldwalk.runonsave`  | 保存文件时自动编译 |
| pug         | `amandeepmittal.pug`     | Pug 语言支持       |
| stylus      | `sysoev.language-stylus` | Stylus 语言支持    |
